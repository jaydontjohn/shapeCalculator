# Shape Calculator (C)

A simple command-line program written in C that calculates:

- Area of 2D shapes
- Surface area of 3D shapes
- Volume of 3D shapes

## Supported Shapes

2D:
- Triangle
- Circle
- Square
- Rectangle

3D:
- Cube
- Sphere
- Pyramid
- Rectangular Prism

## Compile

Using the Makefile:
make

Or manually:
gcc shapeCalc.c -o shapeCalc -lm

## Run

./shapeCalc

Follow the prompts in the terminal to choose the shape and calculation type.
